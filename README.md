### Run on a local host

1. Clone the repository
   ```sh
   git clone https://github.com/cantona94/test-task-team.git
   ```
2. Install NPM packages in the "backend" directory and launch

```sh
npm install
```

```sh
npm run start:dev
```

3. Install NPM packages in the "frontend" directory and launch

```sh
npm install
```

```sh
npm run dev
```
